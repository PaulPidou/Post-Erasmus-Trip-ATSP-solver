Setup server (linux):
save file bachelorthesis_distributedSystem_messaging_starter.conf to /etc/init/
then type 
$ start bachelorthesis_distributedSystem_messaging_starter

to stop:
$ stop bachelorthesis_distributedSystem_messaging_starter




or to manually start the node js server
$ node /var/www/html/bachelorarbeit/socketio/server/server.js
